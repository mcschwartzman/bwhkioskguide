Epic Integration

For some requests, the application contains built-in support for Epic systems. Epic is the standard in Hospital databases and task management. In the code example below, the task in question is the administration of medications. A task is created in the UI and passed through as an endpoint for the Epic server.

Further information about Epic can be found here.

Gradle Install String: [group: 'd18.teamS', name: 'ComputerService', version: '1.1.1']

@FXML
    public void submitReport() {
        // TODO EPIC interaction

        EpicWrapper.send2Epic(ServiceRequest.createService("Medication Report", "Patient's medication report.", Storage.getInstance().getServiceTypesByName("Medical").get(0), null, null, null, true));
        Alert alert = new Alert(Alert.AlertType.INFORMATION);
        alert.setTitle("Service Request Created");
        alert.setHeaderText("Service Request Created");
        alert.setContentText("Your service request was created successfully.");
        alert.showAndWait();

        Main.switchScenes("Nurse View", "/NurseHome.fxml");
    }
