Transport Requests

Transportation request service. Allows for submission of transportation requests to take place at a given time. Can assign requests to employees and can generate reports.

API class is edu.wpi.cs3733d18.teamC.ChonchonTransportationAPI.ChonchonAPI. The database must first be initialized using a list of employees (empty is allowed) before run() can be called:

if (!ChonchonAPI.isDBInitialized())
	ChonchonAPI.initDB(listOfEmployees);
ChonchonAPI.run(0, 0, 1920, 1080, null, null, null);

Gradle install string: [group: 'd18.teamC', name: 'TransportationRequest', version: '1.1.0']

/**
		     * Implementation of Team P's API, which implements a transportation service.
		     */
		    public void onTransportationRequest() {
		        Timeout.stop();
		        TransportationRequest p_api = new TransportationRequest();

		            p_api.run(100, 30, 1000, 700, null, null, null);
		        } catch (ServiceException e) {
		            e.printStackTrace();
		        }
		        Timeout.start();
		    }
