Tech Support

Computer service, allows for registration of device and ticket creation/resolution.

API class is edu.wpi.cs3733d18.SquonksAPI.controller.SquonksAPI. Run with:

SquonksAPI squonks_api = new SquonksAPI();
squonks_api.run(e->OnPathFindClick(e));
private void onPathFindClick(String node_id){ // your pathfinding function call here }

Null can also be passed into the run function.

Gradle Install String: [group: 'd18.teamS', name: 'ComputerService', version: '1.1.1']

@FXML
    Button it_request;

    /**
     * Implementation of our own API, which implements an IT service.
     */
    public void onITRequest() {
        Timeout.stop();
        SquonksAPI squonks_api = new SquonksAPI();
        squonks_api.run(e -> onPathFindClick(e));
        Timeout.start();
    }
