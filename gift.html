Gift Shop Requests

Gift shop. Allows users to pick gifts, add them to a cart, search for gifts, filter gifts by type, etc. Admin actions allow for employee reports, editing/addition of items, and list of transactions.

API class is edu.wpi.cs3733d18.teamOapi.giftShop.GiftShop. Run with run(xCoord, yCoord, windowWidth, windowLength, cssPath, null, null)

Gradle Install String: [group: 'd18.teamO', name: 'GiftShop', version: '1.0.2']

@FXML
			    Button gift_request;

			    /**
			     * Implementation of Team O's API, which implements a gift shop service.
			     */
			    public void onGiftRequest() {
			        Timeout.stop();
			        GiftShop o_api = new GiftShop();
			        o_api.run(100, 30, 1000, 700, null, null, null);
			        Timeout.start();
			    }
