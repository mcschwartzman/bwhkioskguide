Service Requests

Internal Services

Standard Internal Service Requests are handled through an SQL database that stores each request for viewing by Administrative users later. The createService function shown below takes in a user input String for title, a Node as a location to be serviced, and a secondary User as the desired fulfiller.

public static ServiceRequest createService(String title, String description, ServiceType type, User requester, Node location, User desired_fulfiller, Boolean to_epic) { ServiceRequest sr = new ServiceRequest(title, description, type, requester, location, desired_fulfiller); if(to_epic) { EpicWrapper.send2Epic(sr); } return sr; }
