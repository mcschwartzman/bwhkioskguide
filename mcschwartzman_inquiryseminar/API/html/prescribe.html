Prescribe Request

Prescriptions API, allows for creation and deletion of prescriptions assigned to patients. Admin view allows creation, editing, and deletion of employees. Can input data on patient allergies and medical information. Also includes native printer support with any standard receipt printer.

API class is edu.wpi.cs3733d18.teamR.RaikouAPI. Run with run(xCoord, yCoord, windowWidth, windowLength, cssPath, null, null).

Gradle Install String: [group: 'd18.teamR', name: 'Prescriptions', version: '1.0.0']

@FXML
    Button prescription_request;

    /**
     * Implementation of Team R's API, which implements a prescription service.
     */
    public void prescriptionRequestClick() {
        Timeout.stop();
        RaikouAPI raikouAPI = new RaikouAPI();
        try {
            raikouAPI.run(100, 30, 900, 600, null, null, null);
        } catch (edu.wpi.cs3733d18.teamR.ServiceException e) {
            e.printStackTrace();
        }
        Timeout.start();
    }
