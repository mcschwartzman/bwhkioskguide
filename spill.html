Custodial Requests

Spill request service, allows sending of spill requests with priority, status, and assignees. Can mark spills as hazardous and can generate report. Assignees can receive text messages about spill requests and reply to them to update their status.

To run, you'll need to install NPM on your computer; instructions for this can be found here. Once that's installed, install localtunnel using npm install -g localtunnel. After that, you'll need to create a Twilio account; sign up for a trial here and create your own Twilio phone number. Make sure to add all personal phone numbers that you would like to receive texts, using the "verified caller IDs" page on the phone numbers tab.

After this setup is complete, send an email to arroane@wpi.edu to receive a customized jar file. Include your account SID, authorization token, twilio phone number, and all other verified caller ID numbers, and Team B will respond with a customized file within a few hours.

API class is SpillRequest.SpillRequest.
